# PyChan-Bot

## Installing requirements:  
```python -m pip install -r req.txt```

## Configuration:  
### Linux
```cp config.py.sample config.py```  
```nano config.py```

### Windows
copy `config.py.sample` and rename it to `config.py`  
edit `config.py` and fill it out

## Running the bot:  
```python main.py```
